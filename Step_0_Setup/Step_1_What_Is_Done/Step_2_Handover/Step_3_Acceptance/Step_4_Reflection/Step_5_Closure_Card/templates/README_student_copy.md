# 專案結案 Workbook（學生版）

> A step-by-step guide for students to finish a project, not just submit it.

## 使用方式
1. 依序完成 Step_0 到 Step_5 的檔案
2. 每一份檔案都以「能交接給別人」為目標撰寫
3. 完成後可在 README 補上成果連結（Demo、影片、簡報、程式碼、文件）

## 檔案清單
- Step_0_Setup/00_my_project.md
- Step_1_What_Is_Done/01_what_we_did.md
- Step_2_Handover/02_is_it_clear.md
- Step_3_Acceptance/03_is_it_ok.md
- Step_4_Reflection/04_lessons_learned.md
- Step_5_Closure_Card/05_project_closure_card.md
